## Getting Started

1. Clone this template
2. Install dependencies:
   ```sh
   bun install
   ```
3. Start development:
   ```sh
   bun run dev
   ```
4. Build your package:
   ```sh
   bun run build
   ```
5. Publish your package:
   ```sh
   bun run local-release
   ```
